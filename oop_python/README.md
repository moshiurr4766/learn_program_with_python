# OOP Python

Beginner-friendly Object Oriented Programming examples in Python.

Run any file like this:

```bash
python oop_python/01_class_and_object.py
```

## Concepts

1. `01_class_and_object.py` - class, object, attributes, methods
2. `02_constructor_and_self.py` - `__init__` constructor and `self`
3. `03_instance_class_static_methods.py` - instance, class, and static methods
4. `04_encapsulation.py` - public, protected, private style attributes
5. `05_getter_setter_property.py` - property getter, setter, and validation
6. `06_inheritance.py` - parent and child classes
7. `07_multiple_inheritance.py` - inheriting from more than one class
8. `08_polymorphism.py` - same method name, different behavior
9. `09_abstraction.py` - abstract base class
10. `10_composition.py` - one class using another class
11. `11_magic_methods.py` - special methods like `__str__` and `__add__`
12. `12_dataclass_example.py` - shorter class syntax with `dataclass`
13. `13_class_variable_vs_instance_variable.py` - shared vs object-specific data
14. `14_custom_exception.py` - custom exception in an OOP style

## Simple Projects
